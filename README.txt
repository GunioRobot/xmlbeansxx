-=-=-=-= Compiling =-=-=-=-
Required packages for bootstrap invocation:
1. autoconf
2. automake
3. libtool

Because of fact that these packages ofter are not
backward compatible, here are versions known to 
be working:
autoconf: 2.59a
automake: 1.9.6
libtool: 1.5.6

Compiler:
g++-3.3, g++-4.0
